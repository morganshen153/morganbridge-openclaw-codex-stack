# MorganBridge Stack

Third-party integration bundle for routing work from OpenClaw to a connected Codex session through MorganBridge.

This bundle is maintained by `morganshen153`. It is not affiliated with, endorsed by, sponsored by, or certified by OpenAI or the OpenClaw project. References to OpenClaw and Codex describe compatibility only.

## Recommended Public Repository Name

- `morganbridge-stack`

## Component Names

- `morganbridge-runtime`
- `morganbridge-openclaw-plugin`
- `morganbridge-receiver-skill`
