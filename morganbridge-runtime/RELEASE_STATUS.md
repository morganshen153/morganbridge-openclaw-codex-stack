# MorganBridge Runtime Release Status

## Current Classification

`morganbridge-runtime` is currently a bundled binary runtime for the public stack, not a clean source release.

## What This Means

- Do not describe this folder as fully open source in its current form.
- Do not place an Apache-2.0 `LICENSE` file here unless the runtime source and bundled dependency provenance are published with it.
- Treat `MorganBridge.exe`, bundled DLLs, and bundled `.pyd` files as release payloads with separate review requirements.

## Source Boundary

See the repository-level `SOURCE_BOUNDARY.md` for the current release decision and the local-source evidence that was found during audit.
