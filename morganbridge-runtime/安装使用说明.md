# MorganBridge Runtime 安装使用说明

## 这是什么

`morganbridge-runtime` 是包含 `MorganBridge.exe` 及其配套依赖的本地运行时包。

它的作用是：
- 接收任务
- 转交给 Codex
- 接回结果
- 支持继续上一轮对话

## 适合谁用

适合已经在本地使用：
- OpenClaw
- Codex

并且希望把任务从 OpenClaw 交给 Codex 处理的人。

## 安装方式

通常不建议单独裸用它，最推荐的方式是：

1. 把它和 `morganbridge-openclaw-plugin` 插件一起保留
2. 让 OpenClaw 通过插件去调用它

如果你只是保存文件，请确保：
- `MorganBridge.exe` 不被移动到临时目录
- 和它配套的插件配置能找到它

## 怎么用

一般使用方式不是手动双击，而是：

1. 在 OpenClaw 里装好 `morganbridge-openclaw-plugin`
2. 在 OpenClaw 中触发“交给 Codex”
3. 插件自动调用 `MorganBridge.exe`
4. Codex 回来的结果自动返回给 OpenClaw

## 使用建议

- 不要把它当普通聊天程序使用
- 不要随意重命名或拆散配套文件
- 如果你要迁移环境，优先整套迁移，而不是只拷这一个文件

## 一句话理解

`morganbridge-runtime` 负责真正把任务送过去，再把结果带回来。`
