# MorganBridge OpenClaw Plugin 安装使用说明

## 这是什么

`morganbridge-openclaw-plugin` 是第三方 OpenClaw 插件。

它的作用是：
- 在 OpenClaw 里提供“把任务交给 MorganBridge”的实际入口
- 调用 `MorganBridge.exe`
- 把结果带回当前对话

## 安装位置

建议把整个文件夹复制到：

`%USERPROFILE%\.openclaw\extensions\morganbridge-openclaw-plugin\`

不要只拷单个文件，保持整个目录结构完整。

## 安装后要做什么

1. 确认插件目录完整存在
2. 重启 OpenClaw
3. 重新打开聊天或新开一个会话

## 怎么使用

安装后，用户在 OpenClaw 里表达类似意思时，就应该由它参与工作：
- “把这个交给 Codex”
- “用 Codex 继续这个任务”
- “让 Codex 来做”

## 成功效果

正常情况下，你会看到：
- OpenClaw 不再只是口头说“可以交给 Codex”
- 而是能真正把任务发出去
- 并把 Codex 的结果拿回来

## 适合的使用场景

- coding 任务
- 持续处理型任务
- 需要多轮延续的任务

## 一句话理解

`morganbridge-openclaw-plugin` 是 OpenClaw 端真正把桥接动作发起来的那一层。`
